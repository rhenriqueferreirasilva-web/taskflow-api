# TaskFlow API

API REST desenvolvida em C#/.NET para gerenciamento de tarefas, criada como projeto de portfólio back-end.

## Sobre o projeto

O TaskFlow API simula uma aplicação simples de controle de tarefas, com foco em organização, produtividade e acompanhamento de atividades.

O projeto foi estruturado para demonstrar fundamentos de desenvolvimento back-end, separação de responsabilidades e boas práticas de organização de código.

## Tecnologias utilizadas

- C#
- .NET
- ASP.NET Core Web API
- Git/GitHub

## Conceitos aplicados
